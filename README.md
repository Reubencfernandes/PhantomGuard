# PhantomGuard
 The official moderation bot of PhantomDEV

## Dependencies
>- Discord.js 11.5.1
>- eslint 6.3.0

## devDependencies
>- nodemon 1.19.2

## How to use
>- Use "npm run dev" to run in dev mode
>- Use "npm run start" to start the bot
